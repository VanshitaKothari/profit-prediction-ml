# Profit Prediction using Machine Learning

## Overview
This project builds machine learning models to predict profit based on retail sales data.

## Dataset
Superstore dataset containing sales, discount, quantity, and regional information.

## Models Used
- Linear Regression
- Random Forest Regressor

## Approach
- Data preprocessing and feature selection
- Model training and evaluation using R² score
- Feature importance analysis

## Results
The models achieved moderate performance. Profit prediction is complex and influenced by external factors not present in the dataset.

## Key Insights
- Sales and discount significantly impact profit
- Product categories and sub-categories influence profitability
- Machine learning performance depends heavily on feature quality

## Conclusion
This project demonstrates practical application of regression models and highlights real-world limitations of predictive modeling.
